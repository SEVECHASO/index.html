HTML Básico

Bienvenido a este tutorial sobre HTML Básico. Aquí aprenderás todo lo necesario para crear la estructura básica de una página web.

Introducción

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Utiliza etiquetas para definir elementos como títulos, párrafos, enlaces, imágenes y más.

Estructura básica de un documento HTML

Un archivo HTML básico se compone de las siguientes partes principales:

  • <!DOCTYPE html>: declara el tipo de documento.
  • <html>: etiqueta raíz que encierra todo el contenido.
  • <head>: contiene metadatos, el título y enlaces a recursos.
  • <body>: contiene el contenido visible de la página.

Ejemplo sencillo

<!DOCTYPE html>
<html>
  <head>
    <title>Mi primera página</title>
  </head>
  <body>
    <h1>Hola Mundo</h1>
    <p>Este es un párrafo.</p>
  </body>
</html>

Etiquetas comunes

  • <h1> a <h6>: títulos de diferentes niveles.
  • <p>: párrafos.
  • <a href="url">: enlaces.
  • <img src="imagen.jpg" alt="Descripción">: imágenes.
  • <ul>, <ol>, <li>: listas.

Conclusión

Con estos conceptos básicos de HTML puedes empezar a crear páginas web simples y entender cómo estructurar el contenido.
